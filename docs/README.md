# SDLC Hub — Documentation

## Overview

SDLC Hub is a unified visibility and governance platform for the full Software Development Life Cycle — from requirements through code, test, deployment, and production operations. It integrates with existing tools (Jira, GitHub, GitLab, CI/CD, SonarQube, SAST, PagerDuty) to surface actionable insights, enforce quality standards, and close the feedback loop between delivery and operations.

**Tech stack**: NestJS (Node.js) backend + React 19 SPA frontend + PostgreSQL (Prisma ORM) + Tailwind CSS v4.
